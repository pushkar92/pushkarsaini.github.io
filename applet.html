Applets

APPLET

Applet is a Java class file which is embedded in webpage.It is used for displaying graphics on web browser.

AWT


AWT - Abstract Window Toolkit

In AWT each component like button , text field , checkbox , dialogbox etc have their own window as far as the underlying OS(operating system) is concerned , which means large usage of system resources.

AWT are called heavyweight components because of the significant usage of system resources.

Swing components are called lightweight because they are just drawn, they don't need their own windows.

Swing is based on AWT so you need to have thorough knowledge of AWT.
