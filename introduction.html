Introduction to Java

What is Java ?


Java is an object oriented programming language that connect users with information from web servers , databases , and information provider.


History of Java

In 1991 , group of Sun engineers was led by Patrick Naughton and James Gosling. They wanted to design small computer programming language that could be used for devices - cable TV , switchboxes etc.

James named the langauge - "Oak" . But later Sun enginners realised that Oak is already name of existing computer language , that's why they changed the name to Java. In 1994 , Sun engineers thought to build a real cool browser. HotJava browser was written in Java to show the power of Java now called as applets. This technology was shown at SunWorld on 23 , May , 1995 . Sun released the first version of java in early 1996.


It is divided into four core technologies

  1. Java Standard Edition (Java SE)

  2. Java Enterprise Edition (Java EE)

  3. Java Mobile Edition (Java ME)

  4. Java Fx

Features of Java


  1. Simple and Sober

  2. Object Oriented

  3. Portable

  4. Platform Independent or Write Once Run Anywhere

  5. Multi Threaded

  6. Secure

  7. Robust

    1. Java provides built-in tool called Garbage Collector to de-allocate un-used memory to avoid memory leakage

    2. Java provides special classes called exception to exact report the problems in runtime.

    3. Java provides a big set of classes called Java API for almost any purpose



Writing first Java Program


  • A java program file must have .java extension

  • Every Java program must have some set of classes using class keyword


class <classname>

{

//members

}

  • To create an execute class it must have an entry point main()

    • public static void main(String args[])

    • public static void main(String… args)

Syntactical Rules of Java


  • All keywords and package names must be in lower case

  • All class names and interfaces must starts with capital letter

    • Math

    • String

    • BufferedReader

    • InputStreamReader

  • All field and methods starts with lower letter

    • length()

    • readLine()

    • substring()

    • println()


General Input/Output in Java


Java provides built-in static objects under System class

  • in

  • out

  • err


in refers to the standard input (keyboard)

out refers to the standard output (monitor) for general messages

err refers to the standard error (monitor) for error messages


in is a static object of InputStream class from java.io package

out and err are a static objects of PrintStream class from java.io package


Methods of PrintStream class


  • print()

  • println()

  • printf()

What is PATH?


An environmental variable to manage list of folders to lookup executable files (.exe, .com and .bat).


To add a folder temporarily from command prompt issue the following command


PATH=%PATH%;C:\Program Files\Java\jdk1.8.0\bin


To make the permanent PATH setting use


My Computer -> Properties -> Advanced -> Environmental Variables -> System Variables ->Path -> Edit


Paste the path of your BIN after a semicolon
