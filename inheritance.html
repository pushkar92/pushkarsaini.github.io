Inheritance

What is Inheritance ?


INHERITANCE is used to derive one class from another class.

It provide re-usability of code.

 

class A       // A is superclass or parent class or base class

       {         }

class B extends A     // B is subclass or child class or derived class

     {             }

note : -   1. Java provide only single inheritance.

                2. All classes in Java are child of Object class.

 

LET'S TAKE A LOOK AT PROGRAM

( read comments in the program for better understanding. )

 

class ParentA
{
    int i = 10;
    int xyz=70;
    public void printNames()
    {
        System.out.println("Saini " +i);
    }
}
 
class ChildB extends ParentA
{
    //Override
    public int i = 20;
    public int j=100;
    public void printNames()
    {
        System.out.println("Poornima "+i);
    }
}
 


public class MainClass
{
    
    
    //Rule 1 : A parent can hold reference to its child
    //Rule 2 : But a parent can access only those elements that are inherited from it.
    //Rule 3 : The overridden method will be called from Parent's reference.
    //Rule 4 : Cannot reduce the visiblity or scope of the inherited method from Parent class
    // Rule 3 and Rule 4 are for methods only
    // We use extend keyword to inherit a class
    public static void main(String[] args)
    {
        ParentA a = new ParentA();
        ChildB b=new ChildB();
        ParentA c=new ChildB();
    //Reference=Instance
        System.out.println("Parent i value is "+a.i);
        System.out.println("Child i value is "+b.i);
        System.out.println("Child i value with Parent reference is "+c.i);
       c.printNames();
       
    }

}

 

OUTPUT IS : - >


Parent i value is 10
Child i value is 20
Child i value with Parent reference is 10
Poornima 20
